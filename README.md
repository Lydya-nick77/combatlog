# History
Battlemod created by Byrth on Windower. 
Simplelog is a port of Battlemod to Ashita by Spiken.
  

## Combatlog
CombatLog is a SimpleLog-derived addon that parses combat packets and shows output in a dedicated, windowed combat log.  
The addon includes an in-game configuration menu and supports a separate combat log window toggle.

## Features
- Custom combat message formatting
- Condensed target and damage lines
- Message filters by category
- Color profiles for message classes
- Crafting result preview output
- In-game configuration menu
- Windowed combat log rendering

## Commands
- /combatlog
- /clog

### Command actions
- /combatlog or /clog: Open the configuration menu.
- /combatlog log or /clog log: Toggle the combat log window show or hide.

## Notes
- The addon suppresses battle messages from the default FFXI chat window.
